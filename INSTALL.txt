To compile and install the j_py_sad_correlation module, run the following commands:
	git submodule init
	git submodule update
	python setup.py build <args>
	python setup.py install

To build on intel platforms, <args> should be replaced with: --compiler-flags -mssse3
  (or, actually, it may well default to that and therefore no compiler flags are required)
To build on ARM platforms, <args> should be replaced with: --compiler-flags "-mfpu=neon"
